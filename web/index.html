Shopnow Unified Service

Un panel para clientes, productos, inventario y pedidos.

Esta version deja tu contrato listo para Render como un solo servicio. Conserva las APIs base, agrega campos faltantes utiles del repo de referencia y monta una interfaz administrativa para operar todo desde el mismo lugar.

01

Acceso

Registro, login y token en la misma pantalla para entrar rapido al flujo completo.

02

Gobierno

Clientes y productos ahora manejan estado activo, y pedidos muestran mejor contexto.

03

Operacion

Inventario, pedidos y catalogo quedan conectados dentro de una sola API lista para desplegar.
